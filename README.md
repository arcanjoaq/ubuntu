## Dotfiles

This project is an installation script that allows you the installation of programming languages, binaries and utilities in one command!!!

### Supported Operating Systems

* Ubuntu
* Linux Mint

### Installation

The following example shows how to use the installation script:

```sh
curl -s 'https://raw.githubusercontent.com/arcanjoaq/dotfiles/master/install' | bash
```

### What does the script install?

*Install.sh* installs the following tools:

* Terminator
* Meld
* Docker CE
* Docker-Compose
* Java
* Node.js
* Maven
* Micro editor
* Vim
* Minikube
* Kubectl
* Helm 3
* Code
* AWS CLI
* psql
* mongodb-org-shell (mongo) and mongodb-org-tools (mongoimport, mongoexport, mongodump and mongorestore)
* Gradle
* Golang
* Virtualbox
* L2TP
